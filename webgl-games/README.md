# WebGL Games Directory

Эта папка содержит WebGL сборки ваших Unity игр для размещения на сайте.

## Структура папок:

```
webgl-games/
├── game1/              # Первая игра
│   ├── index.html      # Главная страница игры
│   ├── Build/          # Сборка Unity WebGL
│   └── TemplateData/   # Ресурсы шаблона
├── game2/              # Вторая игра
└── game3/              # Третья игра
```

## Как добавить новую игру:

1. Создайте папку с названием игры в `webgl-games/`
2. Экспортируйте игру из Unity в формате WebGL
3. Скопируйте все файлы сборки в папку игры
4. Обновите ссылки в `index.html` основного сайта

## Шаблон для новой игры:

Скопируйте содержимое папки `game1/` как основу для новой игры.
